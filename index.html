1. create a directory named MyFirstRepo 2. cd to that dir 3.run the command 'git init' 4. now, our this dir is monitored by git version control system 5. run the 'git status' to check the status of the folder 6. created a file 'index.html' 7. run 'git add .' or 'git add git commit -m "first commit"/"meaningful message" command to save the file as version control system 9. run "git log" to see the commit list
